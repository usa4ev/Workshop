Задание:
Уровень 1 (нужно сделать): Напишите тест функции ЗначениеПоНаименованию() модуля менеджера справочника «Предопределенные значения». Тест должен проверять, что функция возвращает значение реквизита "Значение" элемента с наименованием, переданным в параметре.
Уровень 2 (желательно сделать): Проверьте, что метод не выбрасывает исключение, когда передано наименование несуществующего элемента.
Уровень 3 (сверхрезультат): Проверьте, что метод выбрасывает исключение с определенным текстом, когда в качестве параметра передана не строка.

Критерий минимального результата:
1. Написан написан тест из по крайней мере одного варианта, который выполняется успешно. 
2. В данном задании не обязательно использовать конструктор вариантов.
3. В данном задании обязательно использовать автоматическое удаление тестовых данных. Тестовые данные должны создаваться с помощью функциональности YAXUnit.
